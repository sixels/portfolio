PWDeck

Rust

PWDeck (Password Deck) is a password manager with a simple CLI interface. You can create a deck (i.e. a password vault) and store all your passwords in there. I learned about crytography, which is an extensive subject, and I want to study more about it soon. PWDeck uses the scrypt encryption alghorithm, which gives it security and safety.
